
INTRODUCTION					*erlang-motions*

Provides key mappings for motions that jump over Erlang functions, function
clauses, and function declarations.

1. Motions					*erlang-motions-motions*

						*]]*
]]			go to next function declaration

						*[[*
[[			previous function declaration

						*]m*
]m			next function clause

[m			previous function clause

And more: `]M`, `[M`, `][`, `[]` go to **end** of next/previous clause/declaration

2. Text objects					*erlang-motions-text-objects*

						*im*
im			inside function clause.

						*am*
am			around function clause.

						*iM*
iM			inside function declaration. `vaM`:

						*aM*
aM			around function declaration. `vaM`:
